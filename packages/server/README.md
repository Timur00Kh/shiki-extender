# Shiki Extender Server

Node.js сервер для Shiki Extender.

## Установка

```bash
npm install
```

## Запуск

```bash
npm start
```

Сервер запустится на порту 81.

## Структура

- `shiki_ex_server.js` - основной файл сервера
- `modules/` - модули сервера
- `db_backups/` - резервные копии базы данных
